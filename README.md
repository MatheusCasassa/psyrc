# RASCS

EN
RASCS is a Streamlit application designed to be a "scoring calculation system" for WCA competitions, based on results published on WCA Live. It allows you to use data posted in real time and score it dynamically. We use as a basis the calculation created by APS (Associacao Portuguesa de Speedcubing), which can be consulted at this link:

https://www.instagram.com/p/DS-H9wJiAoi

This initiative came from Cubing SP (Associação de Speedcubing São Paulo) and was developed by Matheus Casassa.

PT
RASCS é uma aplicação feita em Streamlit onde seu objetivo é ser um "sistema de cálculo de pontuação" para competições da WCA, baseado nos resultados publicados no WCA Live. Com ela, é possível utilizar os dados postados em tempo real e pontua-los de forma dinâmica. Utilizamos como base o cálculo criado pela APS (Associacao Portuguesa de Speedcubing), e que pode ser consultado neste link:

https://www.instagram.com/p/DS-H9wJiAoi

Essa iniciativa partiu da Cubing SP (Associação de Speedcubing São Paulo) e foi desenvolvido por Matheus Casassa
